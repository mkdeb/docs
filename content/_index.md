# What is mkdeb?

mkdeb is an open source application that aims to provide a convenient way to generate Debian packages from upstream
release archives.

## Documentation

This documentation is, as the [mkdeb][project-url] project, currently under development and the various
information provided here may be subject to changes in the near future. You can help making it better by opening an
issue on [Github][project-issues] if you spot an error or if you think of any improvement.
